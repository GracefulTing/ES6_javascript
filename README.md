### 第一课 环境部署

分支提交代码：
    git branch 分支名
    git checkout 分支名
    git push  -u 'https://github.com/GracefulTing/ES6_javascript.git'

1.Visual Studio Code安装：官网下载 -> 打开工具 -> 扩展 -> live-server安装(保存自动看到效果) -> javascript code安装

2.查看命令：ctrl+k,ctrl+s 
  代码放大：ctrl + =                        代码缩小：ctrl + -

3.写入代码保存后右键 Open With Live Server打开网页
